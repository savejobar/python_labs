# Python Learning Labs (Python)

## Описание

Данный репозиторий содержит набор учебно-лабораторных работ, посвящённых  
**изучению и практическому применению языка Python** и его библиотек.

Основная цель проекта — **ознакомление с базовыми и продвинутыми концепциями Python**,  
работа с библиотеками для анализа данных и визуализации, а также создание веб-приложений.

**Используемые технологии:** Python, Jupyter Notebook, NumPy, Pandas, Matplotlib, Seaborn, scikit-learn, Flask, FastAPI

---

## Рассматриваемые задачи

### Основы Python

- Типы данных и структуры данных
- Функции и lambda-функции
- Объектно-ориентированное программирование (OOP)
- Основные концепции Python

### Работа с библиотеками для анализа данных

- **NumPy** — работа с массивами и векторными операциями
- **Pandas** — обработка и анализ таблиц и данных
- **scikit-learn (sklearn)** — базовые алгоритмы машинного обучения

### Визуализация данных

- Построение графиков с помощью Matplotlib и Seaborn
- Настройка визуального отображения данных

### Веб-разработка на Python

- Работа с библиотекой **Flask** для создания веб-приложений
- Работа с библиотекой **FastAPI** для создания REST API

---

## Реализации

- `Basic Python Concepts.ipynb` — Основные концепции языка Python
- `Data types.ipynb` — Типы данных

- `Visualization in Python` — Построение графиков и визуализация данных
